# Project Spec: FocusFlow Todo App

## Overview

FocusFlow is a mini Todo App that allows users to manage a simple task list in the browser. The project is built as a static web app and deployed using GitHub Pages.

## Target Users

Students, learners, and everyday users who need a quick way to organize small tasks.

## Core Requirements

- Users can add a task.
- Users can mark a task as complete.
- Users can delete a task.
- Users can filter tasks by all, open, and done.
- Users can clear completed tasks.
- Tasks remain saved after refreshing the page.
- The app works on mobile and desktop screens.

## User Interface

The interface includes:

- A clear app title and short description
- Task summary counters
- A task input form
- Filter controls
- A task list
- An empty state message

## Technical Requirements

- Built with HTML, CSS, and JavaScript
- Uses local storage for persistence
- No external API required
- No build step required
- Deployable from the GitHub Pages root folder

## Success Criteria

- The user can create and manage tasks successfully.
- The app is readable and responsive.
- The repository includes a README file.
- The project is deployed and accessible through GitHub Pages.
