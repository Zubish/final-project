# FocusFlow Todo App

FocusFlow is a simple browser-based Todo App built with HTML, CSS, and JavaScript. It helps users add tasks, mark tasks as complete, filter the list, and save progress using local storage.

## Features

- Add new tasks
- Mark tasks as complete or open
- Filter tasks by All, Open, and Done
- Clear completed tasks
- View task totals
- Saves tasks in the browser with local storage
- Responsive layout for desktop and mobile

## Technologies Used

- HTML
- CSS
- JavaScript
- GitHub Pages

## How to Use

1. Type a task in the input field.
2. Click **Add** to add it to the list.
3. Use the checkbox to mark a task as complete.
4. Use the filter buttons to view All, Open, or Done tasks.
5. Click **Clear Done** to remove completed tasks.
